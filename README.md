# terraform-aws-password-policy

This module creates a strong password policy that
1. Allows users to reset their passwords
2. Requires 30 characters or more, with uppercase, lowercase, symbols and numbers
3. Requires rotating every 30 days
4. Prevents reuse of the previous 24 passwords
